# Violation Detection AI

An end-to-end AI-powered traffic violation intelligence platform.

## Features

- Identifies violations automatically
- Generates court-ready evidence
- Detects repeat offenders
- Predicts violation hotspots
- Provides explainable AI reasoning for every challan

## Architecture

### Stage 1: Image Enhancement Engine

Input Image → Preprocessing Pipeline

### Stage 2: Multi-Object Detection

Detect Vehicle and subparts of traffic objects

### Stage 3: Violation Intelligence Layer

- Helmet Detection
- Triple Riding Detection
- Seatbelt Detection
- Stop-Line Violation
- Wrong-Side Driving
- Illegal Parking

### Stage 4: License Plate Recognition

Vehicle → Number Plate Detection → OCR

### Stage 5: Evidence Generator

Accumulates detection results and generates a complete traffic violation case.

## Innovation #1: Explainable AI Challan

Instead of generic outputs, the system provides proof and reasoning behind every challan generated by the model.

## Innovation #2: Repeat Offender Risk Score

Stores vehicle number plates in a database and generates a risk score based on past violation history.

## Innovation #3: Violation Heatmap Analytics

- Violation hotspots
- Dangerous intersections
- Peak violation timings

## Innovation #4: Tamper & False Positive Verification

Human-in-the-Loop (HITL) implementation:

- If detection accuracy is below a predefined threshold, the case is flagged for manual review.
- Helps reduce false positives and improve system reliability.
